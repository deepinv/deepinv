Change Log#

This change log is for the main branch. It contains changes for each release, with the date and author of each change.

Current#

New Features#

  • Add deepinv.models.WaveletNoiseEstimator and deepinv.models.PatchCovarianceNoiseEstimator for noise level estimation (#1015 by Matthieu Terris)

  • Add deepinv.physics.Scattering physics for non-linear inverse scattering problems (#1020 by Julian Tachella)

  • Add deepinv.physics.Physics.compute_norm() local operator norm computation for non-linear physics (#1020 by Julian Tachella)

  • Add deepinv.models.BilateralFilter model (#997 by Thomas Boulanger)

  • Add distributed computing framework with deepinv.distributed.DistributedContext, deepinv.distributed.DistributedStackedPhysics, deepinv.distributed.DistributedProcessing, deepinv.distributed.DistributedDataFidelity and deepinv.distributed.distribute() factory function. Supports multi-GPU/multi-process execution with physics-based and spatial tiling distribution strategies (#790` by Beno√Æt Mal√©zieux)

  • Add deepinv.loss.metric.CosineSimilarity to the metrics (#944 by Avithal Lautman)

  • New option to initialize 3D networks (DRUNet, DnCNN, DScCP) from pretrained 2D weights (#958 by Romain Vo)

Changed#

  • (Breaking) Make deepinv.physics.BlurFFT compute a true convolution (now) instead of cross-correlation (before). It is now equivalent to deepinv.physics.Blur with padding="circular" (#825 by Minh Hai Nguyen). For even kernel sizes, the output is now shifted by one pixel to the top-left compared to before.

  • Removed eps parameter from deepinv.optim.linear.conjugate_gradient() (#1011 by Julian Tachella)

  • Deprecated verbose_individual_losses parameter in deepinv.Trainer. Individual losses are now always added to logs when multiple losses are present (#928 by Tiberiu Sabau)

  • Deprecate historical attributes in HDF5Dataset (#764 by J√©r√©my Scanvic)

  • Fix ZeroPrior deepinv.optim.ZeroPrior (#1001 by Victor Sechaud)

  • Fix single-disperser CASSI adjointness (#1029 by Andrew Wang)

  • Add deepinv.physics.ComposedPhysics, deepinv.physics.ComposedLinearPhysics to online documentation (#1000 by Romain Vo)

Many of the features in this version were developed by Minh Hai Nguyen, Pierre Weiss, Florian Sarron, Julian Tachella and Matthieu Terris during the IDRIS hackathon.

  • Added a parameterization of the operators and noiselevels for the physics class

  • Modified the Blur class to handle color, grayscale, single and multi-batch images
