load_example#

deepinv.utils.load_example(name, img_size=None, grayscale=False, resize_mode='crop', device='cpu', dtype=torch.float32)[source]#

Load example image from the DeepInverse HuggingFace using deepinv.utils.load_url_image() if image file or deepinv.utils.load_torch_url() if torch tensor in .pt file or deepinv.utils.load_np_url() if numpy array in npy or npz file.

Available examples for name include (see the HuggingFace repo for full list):

Parameters:

  • name (str) – filename of the image from the HuggingFace dataset.

  • img_size (int, tuple[int]) – Size of the image to return.

  • grayscale (bool) – Whether to convert the image to grayscale.

  • resize_mode (str) – If img_size is not None, options are "crop" or "resize".

  • device (torch.device, str) – Device on which to load the image (gpu or cpu).

  • dtype (torch.dtype) – torch dtype to cast the image to.

Returns:

torch.Tensor containing the image.
