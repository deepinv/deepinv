load_example#

deepinv.utils.load_example(name, img_size=None, grayscale=False, resize_mode='crop', device='cpu', dtype=torch.float32)[source]#

Load example image from the DeepInverse HuggingFace.

Uses deepinv.utils.load_url_image() if image file or deepinv.utils.load_torch_url() if torch tensor in .pt file or deepinv.utils.load_np_url() if numpy array in npy or npz file.

Also adds a batch dimension to the image.

Available examples for name include (see the HuggingFace repo for full list):
